# 🧁 Dulce Capricho - Menú Digital Interactivo

Un menú digital moderno, vibrante y optimizado para dispositivos móviles, diseñado exclusivamente para **Dulce Capricho**. Esta "Landing Page" funciona como un catálogo interactivo que conecta directamente a los clientes con el servicio de toma de pedidos.

## ✨ Características Principales

* **Diseño Mágico y Atractivo:** Interfaz de usuario (UI) basada en paletas de colores pasteles, sombras suaves y un efecto "Glassmorphism" (cristal esmerilado) en las tarjetas de los productos.
* **Animaciones CSS Avanzadas:** * Lluvia infinita de "chispas" (sprinkles) en el fondo.
    * Aparición de tarjetas en cascada al cargar la página.
    * Efecto "Glow" (brillo) y rotación sutil al interactuar con los productos.
    * Botón de llamada a la acción (CTA) con animación de pulso infinito.
* **Conversión Directa:** Integración nativa con la API de WhatsApp (`wa.me`) que genera un mensaje predeterminado para facilitar los pedidos al instante.
* **Totalmente Responsivo:** Diseñado pensando primero en el celular (Mobile-First) para una lectura clara de ingredientes y precios.
* **Toques Creativos:** Cursor personalizado en forma de pastelito (🧁) para pantallas de escritorio.

## 🛠️ Tecnologías Utilizadas

Este proyecto fue construido desde cero sin frameworks pesados para garantizar la máxima velocidad de carga:
* **HTML5:** Estructura semántica.
* **CSS3:** Estilos, variables (Custom Properties), Flexbox y `@keyframes` para todas las animaciones.
* **Google Fonts:** Tipografías *Grand Hotel*, *Montserrat* y *Quicksand* para una estética de pastelería boutique.

## 🚀 Cómo ejecutar el proyecto (Uso Local)

Al ser un proyecto de tecnologías web estáticas, no requiere instalación de servidores ni dependencias.

1.  Clona este repositorio: `git clone https://github.com/TU-USUARIO/dulce-capricho.git`
2.  Abre la carpeta del proyecto.
3.  Haz doble clic en el archivo `index.html` para abrirlo en tu navegador predeterminado.
